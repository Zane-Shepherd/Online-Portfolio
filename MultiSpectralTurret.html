Multi Spectral Turret

This project is focused on the design, analysis, and optimization of a multispectral turret for ING Robotic Aviation, with the intent of creating a highly efficient, versatile, and precise turret system capable of operating at different spectral ranges. This project addresses the need for advanced surveillance capabilities in variable environments.

Noteworthy characteristics of the turret include a multi-gimbal system which allows many degrees of freedom of pan and tilt, the implementation of a light weight yet durable frame, the implementation of a system which optimizes the thermal management, a damper system which reduces vibrations, and a payload which fully integrates all necessary sensors, cameras, and other required hardware.
